O template criado está disponível no site 
https://github.com/ICEI-PUC-Minas-PMV-ADS/pmv-ads-2022-2-e1-proj-web-t10-projeto_voluntariado_ads_puc_turma10  e é composto pelos seguintes

● **ONG voluntariada**

● **Seja Voluntário**

● **Atividades**
